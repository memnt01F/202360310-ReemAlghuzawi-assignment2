# AI Usage Report – Assignment 2

## Tools Used & Use Cases

### ChatGPT

ChatGPT was used as the primary AI-assisted development tool during Assignment 2.  
It supported the development process in the following ways:

- **Code Generation:** Assisted in generating JavaScript logic for the project search filter and form validation.
- **Debugging Support:** Helped identify and fix issues related to event handling, element selection, and dynamic content behavior.
- **UI/UX Suggestions:** Provided recommendations to improve interactivity, animations, and user feedback messages.
- **Documentation Support:** Assisted in writing structured README and technical documentation files.
- **Code Review:** Helped review code organization and ensure consistent formatting and readability.

The AI tool was used as a learning assistant rather than a direct replacement for development work.

---

## Benefits & Challenges

### Benefits

Using AI tools provided several advantages during the development of Assignment 2:

- Improved development speed when implementing interactive features
- Helped understand JavaScript event handling and DOM manipulation
- Simplified debugging and error identification
- Improved structure and readability of code
- Provided guidance for implementing user feedback and animations
- Helped maintain clean and organized documentation

### Challenges

Some challenges were encountered while using AI tools:

- Certain AI-generated solutions were more complex than required for the assignment
- Some suggestions needed simplification to match the project scope
- Careful review was required to ensure compatibility with the existing code structure
- Testing was necessary to confirm that suggested solutions worked correctly

These challenges required critical thinking and adjustments before implementation.

## Learning Outcomes

Through using AI tools in this assignment, I learned:

- How to implement dynamic content filtering using JavaScript
- How to validate form input using event listeners
- How to manipulate the DOM to show and hide elements
- How to provide clear user feedback messages
- How to apply simple animations and transitions using CSS
- How to structure JavaScript code in a clean and maintainable way

This process improved both technical skills and problem-solving ability.

## Responsible Use & Modifications

To ensure correctness, originality, and academic integrity, the following steps were taken:

- All AI-generated code was reviewed before implementation
- Code suggestions were modified to match assignment requirements
- Unnecessary complexity was removed to keep the system simple and efficient
- All features were tested manually to verify correct functionality
- The final implementation reflects personal understanding of the code

AI tools were used responsibly as support tools to enhance learning and productivity, not as a substitute for understanding.

## Reflection

Assignment 2 demonstrated how AI can be used effectively to support software development, improve workflow efficiency, and enhance user experience design.
